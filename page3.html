자연환경과 인간

자연환경은 인간 생활을 결정짓는 기반이며, 인간과 자연의 공존을 위해 생태시민 의식이 필요하다

기후별 인간 생활

기후, 지형 등 자연환경은 의식주·산업·문화 등 인간의 생활 전반에 영향을 끼친다.

  • 적도 주변에 분포 / 일사량이 많아 연중 기온 높고 강수량 많음
  • 토양이 척박하여 이동식 화전 농업 실시

지형과 인간 생활

➕ 지형별 생활 상세 (시험 출제 내용)
  • 산지 지역 ⭐시험: 경사 완만한 지형에서 밭농사·가축 사육 / 열대 고산 지대는 저지대보다 연중 온화 → 고산 도시 발달, 독특한 산지 경관 활용 → 관광 산업, 풍부한 지하 자원 → 광공업
  • 평야 지역 ⭐시험: 창장강·갠지스강·라인강 주변 비옥한 토지 → 많은 사람 거주 / 교통로 건설 유리 → 도시 발달 / 상업 시설 → 경제 활동 주요 공간
  • 해안 지역 ⭐시험: 풍부한 자원 → 어업·양식업 / 해안 경관 → 관광·상업 / 원료 수입·제품 수출 유리 → 대규모 항구·산업 단지 → 공업

기후변화의 영향 ⭐시험

🏔️ 북극 빙하 감소

지난 40여 년간 빠른 속도로 축소. 해수면 상승으로 이어짐

🌊 해수면 상승

해안 저지대 침수 위험. 파키스탄 등 기후 재난 이재민 증가

🌧️ 기상 이변

폭우·폭염·가뭄 등 극단적 기상 현상 빈도 증가 / 기온 상승·강수량 변동 → 식량 생산량 감소

🌿 생태계 변화

작물 재배·동물 서식지 변화 / 기후변화 심화 → 생물 다양성 감소

➕ 북극권 개발 사례 (시험 출제 내용)
  • 기후변화로 북극의 빙하 감소 → 다양한 자원 개발 가능해짐
  • 북극의 경제적 가치 높아짐 (석유·천연가스·광물 자원, 항로 개발)
  • 자원 개발을 둘러싼 국가 간 경쟁 심화
📌 안전하고 쾌적한 환경에서 살아갈 권리 ⭐시험
  • 안전하고 쾌적한 환경에서 살아갈 시민의 권리는 인간 존엄성을 보장받기 위해 반드시 필요한 기본권
  • 기후나 지형의 급격한 변화는 시민의 안전권을 위협한다
  • 국가의 역할: 국민의 생명·재산 보호 → 법률 제정 / 재해 예방·복구·지원 → 정책 수립 / 환경 정비 노력
  • 시민의 역할: 안전 교육·환경 교육에 적극 참여 / 자연환경 변화에 관심 가지기 / 피해 시 정부 기관에 즉각적인 복구와 보상 요청 가능

환경 문제와 해결 노력

세계의 주요 환경 문제 ⭐시험

🏜️ 사막화

사막 주변이 사막으로 변하는 현상. 장기 가뭄 + 과도한 방목·개간 → 식량 생산량 감소·생태계 파괴

🌡️ 지구 온난화

화석 에너지 사용 → 온실가스 증가 → 기온 상승. 해수면 상승·기상 이변

🌫️ 대기 오염

화석 에너지 소비 → 오염 물질 증가. 산성비·미세먼지·황사·스모그

☀️ 오존층 파괴

염화 플루오린화 탄소(CFC) 사용 → 오존층 파괴 → 자외선 증가 → 피부 질환·백내장

➕ 추가 환경 문제 (시험 출제 내용)
  • 열대림 파괴 ⭐시험: 열대 지역에서 무분별한 벌채와 경지 개간 → 자연 환경 파괴 → 생물종 다양성 감소 / 적도 주변 열대 기후 지역에서 발생
  • 해양 오염 ⭐시험: 선박에서 유출되는 원유, 바다로 버려지는 쓰레기·오폐수 등 → 다양한 종류의 해양 오염 발생

➕ 개인의 노력 — 생태 시민 (시험 출제 내용)
  • 생태 감수성 키우기: 인간과 자연이 유기적으로 연결되어 있음을 인식, 우리의 행동이 자연에 어떤 영향을 주는지 성찰
  • 생태 전환적 사고: 인간과 자연의 공존과 지속 가능성을 추구하는 사고방식 필요
  • 일상에서 친환경적인 생활 방식 실천 / 그린슈머(환경 보호를 고려하여 소비하는 사람)
💡 탄소 중립이란?
  • 기업·개인이 발생시킨 CO₂ 배출량을 줄이고 흡수량을 늘려 순 배출량 = 0으로 만드는 것
  • 국제 사회 목표: 2050년까지 탄소 중립 실현, 지구 온도 상승 1.5℃ 이내 억제
  • 그린슈머: 녹색(green) + 소비자(consumer) → 환경을 고려하는 소비자

단원 핵심 정리

✅ 시험에 꼭 나오는 포인트
  • 건조 기후 의복: 온몸을 감싸는 헐렁한 옷 (강한 햇볕·모래바람 차단)
  • 냉대 기후: 타이가(침엽수림대) 넓게 분포 → 목재·펄프 산업 발달
  • 기후변화 원인: 화석 에너지 사용 → 온실가스 증가 (인위적 요인)
  • 인간 중심주의: 자연 = 도구적 가치, 인간만 도덕적 고려 대상
  • 생태 중심주의: 자연 = 내재적 가치, 무생물 포함 생태계 전체가 고려 대상
  • 파리 협정: 195개국, 모든 참여국 감축 의무 (교토의정서: 선진국만)
  • 탄소 중립: 순 배출량 0, 2050년 목표
